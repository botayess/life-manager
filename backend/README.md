# Life Tracker Backend

Бұл backend бөлек demo API ретінде қосылды. Flutter app ішінде offline/local режим жұмыс істейді, ал backend папкасы база дерекпен қорғауға көрсету үшін дайын.

## Start
```bash
cd backend
npm install
npm start
```

API: http://localhost:3000
