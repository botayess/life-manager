# Life Tracker Pro

Flutter project for mobile app development. Personal tracker + group/team tracker.

## Demo accounts
- Admin: `admin@life.kz` / `admin123`
- User: `user@life.kz` / `user123`
- Old test: `test123@gmail.com` / `test123`

## Features
- Login/register
- Admin/User interface
- Habits, finance, notes, plans
- Team/group tracker
- Add members as admin/user
- Assign tasks by admin
- Users can update task status
- Search in team tasks
- RU/EN language toggle
- Light/dark theme
- Backend folder with SQLite demo API

## Run Flutter
```bash
flutter pub get
flutter run -d chrome
```

For Android emulator / phone, generate native platform files if they are missing:
```bash
flutter create .
flutter pub get
flutter devices
flutter run
```

## Run backend
```bash
cd backend
npm install
npm start
```
