# Командный план коммитов

## Ветки
- `main` — стабильная версия проекта
- `bota-ui` — Бота: главная страница, дизайн, карточки, адаптив
- `erke-auth-notes` — Ерке: вход, регистрация, заметки, настройки
- `erasyl-finance-team` — Ерасыл: финансы, диаграмма, аналитика команды
- `askhat-habits-plans` — Асхат: привычки, планы, модуль групповых задач

## Стартовые коммиты
```bash
git init
git add .
git commit -m "start: create Life Tracker Pro combined version"
git branch -M main
git remote add origin YOUR_REPOSITORY_LINK
git push -u origin main
```
